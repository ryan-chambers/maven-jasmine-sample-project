Maven Jasmine Sample Project
============================

This project uses the [jasmine maven plugin](https://github.com/searls/jasmine-maven-plugin).

Browser
-------

mvn jasmine:bdd

http://localhost:8234

Using Maven
-----------

mvn clean install
